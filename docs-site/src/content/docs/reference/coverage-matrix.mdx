## Gaps

- Authenticated user WebSocket streams wait for L2 WebSocket auth tests.
- Data API all-market open interest is not exposed by the CLI yet; the current
  command requires `--token-id`.
- The command reference is manually aligned with the CLI tree. A checked-in
  generator is still a documentation tooling gap.
