title: Quick Start

Three realistic workflows to get you productive.

## 1. Find and inspect a market

```bash
# Search for active BTC markets
polygolem discover search --query "btc 5m" --limit 5

# Get full market details
polygolem discover market --id "0xbd31dc8a..."

# Get everything at once — Gamma metadata + CLOB tick size, fee, orderbook
polygolem discover enrich --id "0xbd31dc8a..."
```

Output (JSON):
```json
{
  "market": {
    "question": "Bitcoin up in 5 minutes?",
    "lastTradePrice": 0.52,
    "bestBid": 0.51,
    "bestAsk": 0.53,
    "outcomes": ["Up", "Down"]
  },
  "tick_size": { "minimum_tick_size": "0.01", "minimum_order_size": "5" },
  "neg_risk": false,
  "fee_rate_bps": 0,
  "order_book": {
    "bids": [{"price": "0.51", "size": "100"}],
    "asks": [{"price": "0.53", "size": "50"}]
  }
}
```

## 2. Check if a market is tradable

```bash
# Get tick size (minimum price increment)
polygolem orderbook tick-size --token-id "123..."

# Get fee rate
polygolem orderbook fee-rate --token-id "123..."

# Get spread
polygolem orderbook spread --token-id "123..."

# Get real-time order book
polygolem orderbook get --token-id "123..."
```

## 3. Use as a Go SDK

```go
import "github.com/TrebuchetDynamics/polygolem/pkg/bookreader"

reader := bookreader.NewReader("")
book, err := reader.OrderBook(ctx, "token-id-here")
// book.Bids[0].Price = 0.51
// book.Asks[0].Price = 0.53
```
