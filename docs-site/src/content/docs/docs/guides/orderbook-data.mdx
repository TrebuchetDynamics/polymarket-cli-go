The `polygolem orderbook` family wraps the public CLOB market-data endpoints.
Every command on this page is **read-only** — no API key, no wallet, no risk.

<Aside type="note">
You only need a CLOB token ID to use these commands. Find one with
[`polygolem discover search`](/guides/market-discovery) or `discover enrich`.
</Aside>

## Get the order book

Full L2 depth (top bids and asks):

```bash
polygolem orderbook get --token-id 71321045679252212594626385532706912750332728571942134274583709853755746957851
```

```json
{
  "market": "0x...",
  "asset_id": "71321045679...",
  "bids": [
    { "price": "0.51", "size": "100" },
    { "price": "0.50", "size": "250" }
  ],
  "asks": [
    { "price": "0.53", "size": "75" },
    { "price": "0.54", "size": "180" }
  ],
  "hash": "0x..."
}
```

## Using the Go SDK

If you want best-first orderbook reads from a Go program, prefer the
`pkg/orderbook` package over shelling out to the CLI. If you need broader CLOB
market data such as market lists, price history, tick size, or fee rate, use
`pkg/clob`.

```go
import "github.com/TrebuchetDynamics/polygolem/pkg/orderbook"

reader := orderbook.NewReader("")
book, err := reader.OrderBook(ctx, tokenID)
// book.Bids[0].Price, book.Asks[0].Size, ...
```

See [Go-Bot Integration](/guides/go-bot-integration) for the full SDK story.
