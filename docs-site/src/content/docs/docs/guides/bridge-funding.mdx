The `polygolem bridge` family wraps Polymarket's Bridge API. Use it to discover
supported chains and create a deposit address that the bridge will credit as
pUSD on Polygon.

<Aside type="caution">
For accounts created **after May 2026**, bridging USDC to your EOA is _not_
sufficient to enable trading. New API users must complete the
[deposit wallet onboarding flow](/concepts/deposit-wallets); only pUSD held by
the deposit wallet counts as CLOB buying power. See
[`polygolem deposit-wallet onboard`](/concepts/deposit-wallets).
</Aside>

## Supported assets

```bash
polygolem bridge assets
```

```json
{
  "supportedAssets": [
    {
      "chainId": "8453",
      "chainName": "Base",
      "token": {
        "name": "USD Coin",
        "symbol": "USDC",
        "address": "0x...",
        "decimals": 6
      },
      "minCheckoutUsd": 45
    }
  ]
}
```

## Create a deposit address

The bridge issues a one-shot address scoped to your destination wallet. Send
supported collateral to this address from the chosen source chain, and the
bridge credits pUSD to the destination wallet on Polygon:

```bash
polygolem bridge deposit 0xYourPolymarketWallet
```

```json
{
  "address": {
    "evm": "0x...",
    "svm": "...",
    "btc": "bc1..."
  },
  "note": "Only certain chains and tokens are supported. See /supported-assets for details."
}
```

## Quote and status SDK methods

The public Go SDK also wraps `POST /quote` and `GET /status/{address}`:

```go
client := bridge.NewClient("", nil)
quote, err := client.GetQuote(ctx, bridge.QuoteRequest{ /* ... */ })
status, err := client.GetDepositStatus(ctx, depositAddress)
```

Those endpoints do not have CLI commands yet. Use
[Bridge API](/docs/reference/bridge-api) for the exact request and response
contracts.

## What's next after bridging

The deposit wallet flow is required for all trading:

```bash
POLYMARKET_PRIVATE_KEY="0x..." \
  polygolem deposit-wallet onboard --fund-amount 0.71
```

Read [Deposit Wallets & Signature Types](/concepts/deposit-wallets) for the full
explanation.
